## Setup

Create the conda environment:

```bash
conda env create -f environment.yaml
```

Activate it:

```bash
conda activate pathmc
```

Update after changes to `environment.yaml`:

```bash
conda env update -f environment.yaml --prune
```

## Docs

Preview the docs locally:

```bash
cd docs
quarto preview
```
